# Flappy-Guy

Flappy guy is a funny yet addictive game with many levels that require the business guy to fly through a series of
obstacles. With each level difficulty and coin values increases. Get excited to complete all levels, bet its not easy.
Player has to move through a set of compressor which moves closer to each other.
Guaranteed fun along with smooth play !

So download and start your fly...

Features:
Cool graphics
Simple Control
Fun unlimited

Share your views and have Fun playing
